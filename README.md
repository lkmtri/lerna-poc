# lerna-poc

## Pre-requisites
1. npm i -g yarn
2. npm i -g lerna

## Setup
1. Clone this repo
2. At root folder, run ```lerna bootstrap```
3. ```cd packages/fesc``` and ```yarn dev```, visit http://localhost:3000 on browser

## Features
1. Hot reloading in dev mode.
2. babel module-resolver work properly.
